# 一、课程性质

思想政治课程是高中阶段落实立德树人根本任务的关键课程，本课程旨在用党的创新理论铸魂育人，引导学生广泛践行社会主义核心价值观，是帮助学生确立正确的政治方向、提高思想政治学科核心素养、增强社会理解和参与能力的综合性、活动型学科课程。

高中思想政治课程紧密结合社会实践，讲述马克思主义基本原理，讲述马克思主义中国化时代化成果，特别是习近平新时代中国特色社会主义思想，引导学生经历自主思考、合作探究的学习过程，了解新时代中国特色社会主义经济、政治、文化、社会、生态文明建设和党的建设进程及成就，理解新时代新征程中国共产党的使命任务，培育政治认同、科学精神、法治观念和公共参与等核心素养，逐步树立共产主义远大理想和中国特色社会主义共同理想，坚定中国特色社会主义道路自信、理论自信、制度自信、文化自信，树立正确的世界观、人生观、价值观。

高中思想政治课程具有政治性、思想性、时代性、综合性、实践性，基于大中小学思想政治教育一体化建设的理念，与义务教育道德与法治、高校思想政治理论课等衔接，与时事政治教育相互补充，与高中其他学科教学和相关德育工作相互配合。

# 二、课程理念

一是坚持正确的思想政治方向。本课程立足时代发展，对学生进行马克思主义基本理论教育，用习近平新时代中国特色社会主义思想铸魂育人，培养德智体美劳全面发展的社会主义建设者和接班人，使他们理解马克思主义中国化时代化就是马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合的过程，习近平新时代中国特色社会主义思想实现了马克思主义中国化时代化新的飞跃。面对当前社会变革和实践创新中的新挑战、新问题，要用历史的眼光、国情的眼光、辩证的眼光、文化的眼光和国际的眼光，引领学生通过观察、辨析、反思和实践，真学真懂真信真用马克思主义，立强国报国大志向，做挺膺担当奋斗者。

二是构建以培育思想政治学科核心素养为主导的活动型学科课程。本课程力求构建理论逻辑与实践逻辑、学科体系与生活关切相结合的活动型学科课程。学科内容采取思维活动和社会实践活动等方式呈现，即通过一系列活动及其结构化设计，实现“课程内容活动化”“活动内容课程化”。本课程关注思想政治学科核心素养的培育，坚持教育与生产劳动和社会实践相结合，着眼于学生的真实生活和长远发展，使理论观点与生活经验、劳动经历有机结合，让学生在社会实践活动的历练中、在自主辨析的思考中感悟真理的力量，自觉践行社会主义核心价值观。

三是构建把道理讲深讲透讲活的教学方式。本课程基于高中学生思想认识和行为方式的多样性、可塑性，着力改进教学方式和学习方式。在课程实施中，充分利用现代信息技术，拓展教育资源和教育空间；通过议题的引入、引导和讨论，推动教师转变教学方式，师生在互动、开放、民主的氛围中开展教学；通过创设问题情境，引导学生参与社会实践活动，促进其转变学习方式，在合作学习和探究学习的过程中，培养创新精神，提高实践能力。

四是建立知行合一的综合评价方式。本课程紧紧围绕思想政治学科核心素养的形成与发展，建立激励学生不断发展进步的评价机制。注重学生学习、劳动和社会实践活动的行为表现，采用多种评价方式，综合评价学生的政治认同度、价值判断力、理论思维能力、法治素养和社会参与能力等，全面反映学生思想政治学科核心素养的发展状况。

# 三、课程目标

# （一）学科核心素养内涵

学科核心素养是学科育人价值的集中体现，是学生通过学科学习而逐步形成的正确价值观、必备品格和关键能力。思想政治学科核心素养，主要包括政治认同、科学精神、法治观念和公共参与。

# 1. 政治认同

我国公民的政治认同，就是拥护中国共产党的领导，坚持马克思主义，坚持和发展中国特色社会主义，认同中华人民共和国、中华民族、中华文化，弘扬和践行社会主义核心价值观。

中国特色社会主义是党和人民一百多年奋斗、创造、积累的根本成就，是中国改革开放取得一切成绩和进步的根本原因，是改革开放以来中国共产党的全部理论和实践的主题。社会主义核心价值观是当代中国精神的集中体现，凝结着全体人民共同的价值追求，是凝聚人心、汇聚民力的强大力量。认同中国特色社会主义和社会主义核心价值观，才能形成全国各族人民团结奋斗的共同思想基础，坚持中国道路、弘扬中国精神、凝聚中国力量，为实现中华民族伟大复兴的中国梦而团结奋斗。青少年的政治认同是他们创造幸福生活的精神支柱、价值追求和道德

准则；发展政治认同素养，才能牢固树立中国特色社会主义理想信念，厚植爱国主义情怀，成为德智体美劳全面发展的社会主义建设者和接班人。

# 2. 科学精神

我国公民的科学精神，就是在认识世界和改造世界的过程中表现出来的一种精神取向，能够学习、运用马克思主义的科学世界观和方法论，对个人成长、社会进步、国家发展和人类文明作出正确的价值判断和行为选择。

当代中国正经历广泛而深刻的社会变革，正进行宏大而独特的实践创新。在这一社会变革和实践创新的过程中发扬科学精神，必须坚持辩证唯物主义和历史唯物主义基本观点，领会习近平新时代中国特色社会主义思想的世界观和方法论以及贯穿其中的立场观点方法，理解共产党执政规律、社会主义建设规律、人类社会发展规律，必须坚持解放思想、实事求是、与时俱进、求真务实，一切从实际出发，正确认识和理解新时代改革开放和社会主义现代化建设的实际问题。培养青少年的科学精神，有助于他们形成正确价值取向和道德定力，提高辩证思维能力，立足基本国情、拓展国际视野，在实践创新中增长才干。

# 3.法治观念

我国公民的法治观念，就是树立宪法法律至上、法律面前人人平等、权利义务相统一和依法治国理念，尊法学法守法用法，形成法治信仰和维护公平正义的意识。

建设法治中国，是推进国家治理体系和治理能力现代化的必然要求；全面依法治国，必须坚定不移走中国特色社会主义法治道路，坚持党的领导、人民当家作主、依法治国有机统一，坚持以人民为中心，坚持依法治国和以德治国相结合，全面推进科学立法、严格执法、公正司法、全民守法，在全社会树立法治观念。增强青少年法治观念，有助于

他们依法行使权利、履行义务，尊重自由平等，维护公平正义，弘扬社会主义法治精神，传承中华优秀传统法律文化，做社会主义法治的忠实崇尚者、自觉遵守者、坚定捍卫者。

# 4. 公共参与

我国公民的公共参与，就是有序参与公共事务，勇于承担社会责任，积极行使人民当家作主的政治权利。

广泛的公共参与，彰显人民主体地位，是公民行使知情权、参与权、表达权、监督权的表现，有助于更好地表达民意、集中民智，提高立法和决策的科学性、民主性；有助于鼓励人们热心公益活动，激发社会活力，提高社会治理水平。培养青少年公共参与素养，有益于他们了解民主管理的程序，体验民主决策的价值，感受民主监督的作用，增强公德意识和参与能力，追求更高的道德境界。

# （二）结构

必修课程是培育全体学生学科核心素养的基本载体。选择性必修课程是对必修课程的延展，满足学生多样化的学习兴趣和升学需要。选修课程更关注学生专业素养发展、高校自主招生及学生个性化发展的需要。

以坚持和发展中国特色社会主义为主线，设计必修课程的整体框架，包括四个模块。模块1“中国特色社会主义”，依循历史进程，讲述为何开创和发展中国特色社会主义；模块2“经济与社会”、模块3“政治与法治”、模块4“哲学与文化”，依托模块1的基本原理，讲述如何坚持和发展中国特色社会主义。

选择性必修课程设置“当代国际政治与经济”“法律与生活”“逻辑与思维”三个模块，与必修课程的实施相互配合、相互补充。选修课程设置“财经与生活”“法官与律师”“历史上的哲学家”三个模块，是对相关必修课程和选择性必修课程的进一步拓展。

# （三）学分与选课

必修课程设置四个模块，共6学分；选择性必修课程设置三个模块，共6学分。必修课程与选择性必修课程作为国家课程总计12学分。必修课程是全体学生必须完成的学业。选择性必修课程供对该课程有兴趣的学生选修，计入毕业学分。选修课程涉及个人生活、职业体验、大学先修等方面的内容，可根据学生个性化发展的需求和当地经济、科技、文化发展的特点开设，纳入校本课程管理，由学生自主选择修习。

# 模块3：逻辑与思维

通过科学思维的训练，引导学生掌握科学思维的基本要求，把握逻

辑思维和辩证思维的方法，提高创新思维能力，学会运用科学思维探索世界、认识世界。

1.1 描述常见的思维活动，体会思维是人所特有的属性，了解思维的基本形态和特征；懂得正确思维的基本条件。

1.2 区分抽象思维和形象思维；掌握科学思维的特点，体悟学会科学思维的意义。

# 2. 遵循逻辑思维要求

2.1 知道概念是反映事物本质属性的思维形式；理解任何概念都是内涵和外延的统一。

2.2 知道判断的基本特征；了解形成恰当判断的条件；学会正确运用判断；结合具体的判断活动，区分判断的不同类型。

2.4 辨析常见的逻辑错误，掌握形式逻辑的三个基本规律。

# 3. 运用辩证思维方法

3.1 结合对复杂事物的把握，体会辩证思维的特征；理解分析与综合的辩证关系。

3.2 联系事物发展过程中的渐进性和飞跃性，懂得事物的发展过程是量变与质变的统一；理解质量互变规律；把握适度原则。

3.3 辨析简单肯定一切或否定一切的危害，了解辩证否定观的实质，体会认识不断深化的历程。

# 4. 提高创新思维能力

4.1 理解创新思维的重要性；运用科学思维方法，提高创新思维能力。

4.2 学习运用联想思维，了解发散思维和聚合思维的意义，以超前思维把握事物的发展趋势。

# （三）学业质量标准的应用

学业质量标准可以用来指导学校课程设计、教学和考试评价。思想政治学科学业质量水平2是高中毕业应当达到的要求，也是该学科学业水平考试合格等级的划定依据。思想政治学科学业水平等级性考试命题不超过学业质量水平3。

# 七、实施建议

# （一）教学与评价建议

上好思想政治课，关键在教师。思想政治课教师应发挥积极性、主动性、创造性，按照政治强、情怀深、思维新、视野广、自律严、人格正的要求，不断提高自己的专业素养，坚持政治性和学理性相统一、价值性和知识性相统一、建设性和批判性相统一、理论性和实践性相统一、统一性和多样性相统一、主导性和主体性相统一、灌输性和启发性相统一、显性教育和隐性教育相统一，不断增强思想政治课的思想性、理论性和亲和力、针对性。教师在教学过程中，应以课程标准为依据，以发展学生思想政治学科核心素养为目标，力求将学业质量转化为具体的教学要求，推进“教一学一评”一体化。

# 1. 强化价值引领，把道理讲深、讲透、讲活

思想政治课的本质是讲道理。本课程的教学与评价，必须凸显价值引领的意义，注重方式方法。教师需要用支撑思想政治学科核心素养的基本观点统筹、整合学科知识。可通过范例分析展示观点，在价值冲突中深化理解，在比较、鉴别中提高认识，在探究活动中拓展视野，引导

学生思考和行动，用社会主义核心价值观铸魂育人，促进知行合一。

思想政治学科核心素养的培育，要遵循高中学生认知规律，创新教师教与学生学的方式方法。融合应用现代信息技术，推进基于真实情境的教学；积极采用小组学习、问题解析、学生讲述等课堂形式，注重用好学生身边可知可感的生动事例和典型人物，充分调动学生参与的积极性、主动性。教师要以鲜活的语言、真挚的感情，善于用讲故事的方式，把道理讲深、讲透、讲活，着力增强课堂教学实效，打动心灵、感动学生、入脑入心，培养让党放心、爱国奉献、担当民族复兴重任的时代新人。

在教学中切实强化价值引领，学习路径的选择至关重要。应立足当今信息化环境下学习的新特点，直面社会思想文化的影响相互交织、相互渗透以及学生接受信息的渠道明显增多等新态势；要着眼于学生思想活动的独立性、选择性、多变性、差异性等特点，引导他们步入开放的、辨析式的学习路径，理性面对不同观点。只有使学生亲历自主辨识、分析的过程，并作出判断，才能真正实现有效的价值引领，促进知行合一。

在教学中切实强化价值引领，要注意把握好过程与结论的关系，既关注过程，又不忽略结论；要有效掌控导向性与开放性的关系，取向求同或取向求异，都需要合理引导；要恰当处理思想内涵与辨析形式的关系，遵循意义优先、兼顾形式的原则。

# 2. 围绕议题，实施活动型学科课程的教学

活动型学科课程的实施要使活动设计成为教学设计和承载学科内容的重要形式。一方面，要对应结构化的学科内容，力求提供序列化的活动设计，并贯穿教学全过程；另一方面，要针对相关活动，设计可操作的测评。

教学设计能否反映活动型学科课程实施的思路，关键在于确定开展活动的议题。活动型学科课程的议题，既包含学科课程的具体内容，又

展示价值判断的基本观点；既具有开放性、引领性，又体现教学重点、针对学习难点。围绕议题展开的活动设计，包括提示学生思考问题的情境、运用资料的方法、共同探究的策略，并提供表达和解释的机会。活动设计应有明确的目标和清晰的线索，统筹议题涉及的主要内容和相关知识，并进行序列化处理。要了解学生对议题的认识状况及已有经验，以提高教学的针对性、实效性；还要了解议题的实践价值，创设丰富多样的教学情境，引导学生面对生活中的各种现实问题。

活动型学科课程的教学评价，应专注学科核心素养的行为表现，一般采用“求同”取向与“求异”取向相结合的验证思路。这是一种有统一标准、无标准答案的评价。应以基本观点为统一标准，在此前提下，采用多种活动方式，鼓励学生运用相关学科知识和技能，基于不同经验、运用不同视角、利用不同素材，表达不同见解、提出不同问题解决方案。既评价达成基本观点的过程，也评价实现教学设计的效果。

# 3. 优化案例，采用情境创设的综合性教学形式

本课程内容涉及哲学、经济学、政治学、法学等学科，具有综合性。教学与评价既要体现内容的广泛性，又要关注问题的复杂性；既要多维度观察对象，又要多途径进行探究。应力求凭借相关情境的创设，提供综合的视点，提升综合能力。

根据权威资料，精选和优化案例进行综合性教学，既要着眼于同一课程模块的内容，综合不同的学科核心素养方面，又要着眼于同一学科核心素养方面，综合不同课程模块的内容。精选案例的关键在于：能有效地支持、服务于学科核心素养的培育；有助于呈现并运用学科的核心概念和方法；能充当组织教学内容、贯穿逻辑线索的必要环节；其内在意涵具有丰富的、现实的、可扩展的解释空间；围绕议题，指导、组织富有成效的活动；显现生活中真实的情境，力求可操作、可把握。

实施情境创设的综合性教学评价，重点是考查学生整合知识、理论联系实际、分析和解决问题的能力。进行综合性评价的过程，也是反思

和评估情境创设和案例选取是否得当、是否高效的过程，可据此进一步优化情境、案例，不断提高教学效率和效果。

# 4. 走出教室，把“思政小课堂”同“社会大课堂”结合起来

要坚持理论性和实践性相统一，用科学理论培养人，重视课程的实践性，善用“社会大课堂”，引导学生走出教室、走向社会，将中国特色社会主义伟大实践及时融入教育教学，推动思想政治课教学与学生社会实践活动有机结合。

社会实践活动包括志愿服务、社会调查、专题访谈、参观访问以及各种职业体验等。校外社会实践活动为教学提供了更广阔的空间、更丰富的资源、更真实的情境。开展社会实践活动，要从学生的成长需要出发，用好爱国主义教育基地、红色教育基地、研学教育基地、综合实践基地、法治教育实践基地、文化场馆、科技场馆、博物馆等校外教育资源以及地方特色教育资源，丰富教学内容，加深学生对社会的认识与理解。

社会实践活动的评价可以议题为纽带，以活动任务为依托，不仅要评价有关学科内容的学习效果，而且要评价学生在社会实践活动中表现出来的情感、态度、能力；可以学生的自我记录、自我小结为主，兼顾同学、教师、家人、社区工作人员等的评价。评价的关注点是学科核心素养能否得到提升，具体要看学习目标是否明确，活动设计是否合理，活动组织是否恰当，活动资源是否充分利用，学生的主体性、创造性是否得到充分发挥，学生的交往能力是否得到增强，学生是否有获得感、成就感。

# （二）学业水平考试建议

# 1. 考试理念

$\spadesuit$  坚持正确的政治方向和价值导向。命题要全面贯彻党的教育方

针，落实立德树人根本任务，以社会主义核心价值观铸魂育人，引导学生坚定中国特色社会主义道路自信、理论自信、制度自信、文化自信，树立正确的世界观、人生观、价值观。

$\spadesuit$  坚持核心素养立意。整体考查政治认同、科学精神、法治观念和公共参与等学科核心素养发展状况，强化学科核心素养四个方面的内在联系，关注各方面在学业质量标准不同水平的发展性和同一水平中的关联性。

$\spadesuit$  注重知行合一。增强学科任务的实践性，体现综合运用思想政治学科相关内容解决真实情境问题的要求；赋予学生呈现思维过程与学习成果的空间，让学生的政治立场、价值观、品格和能力得以真实表现；注重考查对中国化时代化的马克思主义的认同和对社会主义核心价值观的践行。

$\spadesuit$  运用多种评价，创新评价方式。根据不同考试性质，改进纸笔测试，强化过程性评价，从时空、载体、组织、技术等方面探索和完善评价手段，使考试评价更加全面客观有效。

$\spadesuit$  促进“教一学一评”有机衔接，形成育人合力。依据学业质量水平解释考试结果，报告学生学业水平达成情况，为评估课程实施和改进教学提供参考。

# 2.考试要素

# （1）考试目标

考试目标主要是考查学科核心素养发展状况。确定具体考试目标，要立意鲜明，凸显思想政治学科的育人价值；全面反映学科核心素养，内容清晰、结构合理；符合学业质量标准要求。

# (2)考试范围

学业水平合格性考试的范围是必修内容，学业水平等级性考试的范围是必修和选择性必修内容。

# （3）考试形式

考试形式指试题或试卷呈现的载体形式以及考生应答、完成任务的条件和行为方式。考试形式的确定要与考试性质、目的以及思想政治学科特点相匹配。学业水平合格性考试一般采用纸笔测试形式，鼓励探索纸笔测试与社会实践相结合等多种科学有效的形式。学业水平等级性考试采用闭卷纸笔测试形式。

# 3. 测试结构

# （1）内容结构

内容比例原则上与各模块课程内容所占学分比例大体相当，可根据实际情况适当调整不同模块所占比例，及时反映马克思主义中国化时代化最新成果。

# （2）题型结构

纸笔测试题型为选择题和非选择题两类，选择题采用结合材料的组合型和非组合型单选题，非选择题可采用结合材料的简答题、论述题、小论文、辨析题等。要根据考试目标，合理确定选择题和非选择题的比例。

# （3）难度结构

考试应涵盖不同难度的试题，要根据考试性质、目的和考生群体特征设计难度结构，覆盖不同难度的试题，合理确定易、中、难题比重。

# 4. 命题蓝图

制定命题蓝图要遵循考试设计要求，具体化试卷结构，整体规划命题，明确每道试题的测试功能和命制要求。命题蓝图旨在落实核心素养命题立意，确保考查内容的代表性和命题的科学性。

命题蓝图要综合考虑试题考查目标、试题特征等因素以及年度考试命题工作新要求，主要包括学科核心素养、学科任务、试题情境、学科

内容、学业质量水平等，综合考虑题型、分值、预设难度、预设作答时间等题目信息。

# 5. 试题命制

试题命制要准确把握学科核心素养内涵和学生学业成就表现关键特征，创设真实情境，设置有助于展现学生不同素养水平的、符合思想政治学科性质的典型学科任务，结合学生真实表现研制评分参考。

# （1）命题指向

遵循考试设计思路，落实命题蓝图要求，明确每道试题所要考查的学科核心素养及其发展水平。一方面，要准确把握学科核心素养与学科任务、试题情境、学科内容之间的内在关系；另一方面，要把握学科核心素养在行为表现上相互交融、逻辑上相互依存，是一个有机整体，避免测试中把核心素养割裂化、碎片化和机械化。

# (2) 情境设置

可从经济、政治、文化、社会、生态文明建设等方面选择情境素材，用文字、数据、图表、漫画等多种形式呈现情境素材，要从情境要素及其关系、情境结构化程度等方面控制情境的复杂程度。试题创设的情境要具有以下特征。

$\spadesuit$  思想性。体现正确的政治方向和价值取向，反映新时代经济社会发展重大成就，引导学生坚定理想信念，树立正确的世界观、人生观、价值观。

$\spadesuit$  真实性。真实可靠、符合逻辑。注重从自然、社会现象和人类认识世界的理论成果中选取素材，加工处理，如实反映社会发展的历史进程，杜绝使用有悖事实和常理的情境材料。

$\spadesuit$  典型性。选取能够集中反映相关领域发展状况、思想政治学科基本立场观点方法，具有代表性的人物、事件、案例、数据等，凸显思想政治学科的特色，能够成为有效测试思想政治学科核心素养发展水平的载体。

$\spadesuit$  适宜性。贴近学生生活实际和思想实际，符合学生身心发展特点和认知发展状况；情境的复杂程度、新颖程度、开放程度要与考试性质相适应。

# （3）任务设计

任务设计应注重思想政治学科思想方法的理解和运用，如观察、描述社会现象，解读分析文字、数据、图表信息，解释、论证或辨析观点，提出意见、建议，设计、评价方案等。任务设计要体现以下要求。

$\spadesuit$  彰显学科特色。任务要反映思想政治学科性质要求，价值导向正确，有助于学生在作答中表现出正确的政治立场和知行合一的品格。

$\spadesuit$  指向明确。任务应从情境中合理生成、自然引出，注重与情境的一致性，任务要求简明、清晰，无歧义，避免任务主题过于宏大或者范围过于宽泛。

$\spadesuit$  凸显基础性。任务应围绕思想政治学科的核心内容展开，着重考查马克思主义基本立场观点方法在不同情境的体现和运用，指向核心素养考查主旨，避免对学科知识的过深过细考查倾向。

$\spadesuit$  注重应用性。任务设计要引导学生学以致用，运用所学马克思主义基本立场观点方法和习近平新时代中国特色社会主义思想的世界观和方法论分析和解决实际问题。

$\spadesuit$  增强综合性。任务设计要根据不同考试的性质和目的，依托复杂情境，加强模块内的内容整合，探索跨模块内容的综合运用，避免考试内容的碎片化以及与模块的机械对应。

$\spadesuit$  体现创新性。任务设计要鼓励学生通过多种方式分析和解决问题，并在迁移和拓展中展现创造性的思维过程。

# （4）评分要求

评分要求应与试题考查目标、设问相一致，符合学业质量标准；要表述准确、无歧义，条理清晰。

选择题应根据命题蓝图的预设分值确定赋分标准。非选择题依据命题蓝图，对照命题指向，研制评分参考，明确分数等级、特征表现、分

数解释等。评分参考应从观点、论据、语言表达、逻辑条理等方面关注学生作答过程的整体表现；不同等级特征表现的描述应准确清晰，具有可操作性。

# （三）教材编写建议

编写好思想政治课教材，要牢牢把握正确政治方向和价值导向，用心打造培根铸魂、启智增慧的精品教材。

# 1. 突出铸魂育人要求，着力培育思想政治学科核心素养

教材编写要坚持以习近平新时代中国特色社会主义思想为指导，立足“培养什么人、怎样培养人、为谁培养人”这个根本问题，用社会主义核心价值观铸魂育人。要坚持正确的政治立场，以培育思想政治学科核心素养为着力点，有机融入重大主题教育内容，处理好政治性和学理性、价值性和知识性、建设性和批判性、理论性和实践性、显性教育和隐性教育等关系，充分体现中国特色社会主义道路自信、理论自信、制度自信、文化自信。

# 2. 遵循课程标准的要求，有利于教师教与学生学

教材编写要以课程标准为依据，充分体现课程标准阐述的基本理念，基于高中学生的认知水平，符合高中阶段学生的特点。要在课程内容与课程结构、素材选择与呈现方式等方面，符合培养学科核心素养的要求，既要有利于教师进行创造性教学，又要有利于学生自主学习和潜能发挥，满足不同类型学生发展的需求。素材选择和活动设计要考虑城乡差异和地区差异，贴近学生生活，提高教材的适用性。

# 3. 利用多种课程资源，拓展学生视野

教材编写要充分、及时反映新时代中国特色社会主义实践和理论创新最新成果，善于融通古今中外各种资源，特别是要把握好马克思主义、中华优秀传统文化和国外哲学社会科学三种资源，要按照立足中国、借鉴国外，挖掘历史、把握当代，关怀人类、面向未来的思路，把中国好的东西坚持好，把国外好的东西借鉴好，使教材既有深厚历史底蕴，又有鲜明时代特点，全面助力培养学生更广阔的眼界、更开阔的思路。

# 4. 体现活动型学科课程实施的新要求，促进知行合一

教材编写要遵循思想政治教育规律、学生成长规律，着力体现活动型学科课程实施的特点。教材设计要有利于课堂教学与课外活动、理论学习与社会实践活动的对接和融合，有利于“大思政课”的展开；要通过设置开放的教学情境，提供多种课内外探究活动，注重发展学生自主学习能力，通过学习、体验和实践，内化于心、外化于行。

# （四）对地方和学校实施本课程的建议

# 1. 加强党对课程实施的全面领导

建立党委统一领导、党政齐抓共管、有关部门各负其责、全社会协同配合的工作格局，推动形成全党全社会努力办好思想政治课的良好局面。学校党委要切实负起主体责任，学校党委书记、校长要带头走进课堂，带头推动思想政治课建设，带头联系思想政治课教师，充分发挥思想政治课立德树人的关键课程作用。

# 2. 加大落实课程实施的教师培训

地方各级教育部门要加强教师专业课程培训。培训要面向全体教

师，重在坚定教师的理想信念，做到“让有信仰的人讲信仰”。在培训中，要加强习近平新时代中国特色社会主义思想的培训，关注大中小学思想政治教育一体化的要求，关注不同学段的内容衔接和侧重点，关注不同课程的相互配合。培训设计要注重整体性，重点阐释思想政治课程的育人价值，促进教师从整体上认识和理解思想政治课程标准。

# 3. 保障思想政治课程的有效实施

地方各级教育行政部门、学校要给予更多资金保障和活动条件支持，确保思想政治课有效开展。学校要开足课时，严格落实普通高中思想政治课程标准必修课程学分要求，积极创造条件开好思想政治课选择性必修、选修课程。地方和学校要重视调动教师资源开发利用的主导作用，同时积极争取社会各方面力量的参与和支持；重视数字化资源开发的信息化环境建设；逐步建立地区、学校之间的资源互补、共建共享机制。
